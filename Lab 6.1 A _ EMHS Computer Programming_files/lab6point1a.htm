Tracking Sales – Lab6point1A

File Sales.java contains a Java program that prompts for and reads in the sales (Dollar Amount) for each of 5 salespeople in a company. It then prints out the id and the dollar amount of the sales for each salesperson and the total dollar amount of the sales for all the salespeople combined. Study the code, then compile and run the program to see how it works. Now modify the program as follows:

1.     Compute and print the average sales in dollars. (You can compute this directly from the total; no loop is necessary.)

 

2.     Find and print the maximum dollar amount of all the individual sales.  Print both the id of the salesperson along with the maximum amount (in Dollars) of the sales, e.g., "Salesperson 3 had the highest sales with $4500." Note that you don't need another loop for this; you can do it in the same loop where the values are read and the sum is computed.

 

3.     Do the same for the minimum dollar amount of all the individual sales.

 

4.     After the list, sum, average, max and min have been printed, ask the user to enter a “Threshold” value.  Then print the id of each salesperson who exceeded that amount, and the dollar amount of their sales. Also print the total number of salespeople whose sales exceeded the “Threshold” value entered.

 

5.     The salespeople are objecting to having an id of 0 -- no one wants that designation.  Modify your program so that the ids run from 1-5 instead of 0-4. Do not modify the array – e.g. print the information for salesperson 1 by using the data in array location 0, and so on.

 

6.     Instead of always reading in 5 sales amounts, at the beginning ask the user for the number of salespeople and then create an array that is just the right size. The program can then proceed as before.

// Sales.java

//

// Reads in and stores the dollar amount of the sales for each of 5 salespeople.

// Displays the dollar amount of the sales entered by salesperson id and total sales for all salespeople.

//

// *********************************************************************************

 

public class Sales

{

    public static void main(String[] args)

    {

            final int SALESPEOPLE = 5;

            int[] sales = new int[SALESPEOPLE];

            int sum;

 

            for (int i=0; i<sales.length; i++)

                {

                        sales[i] = GetInfo.getInt ("Enter the dollar amount for the sales generated by salesperson " + i + ": ");

                }

 

            System.out.println("\nSalesperson        Dollar-Amount");

            System.out.println("--------------------");

            sum = 0;

            for (int i=0; i<sales.length; i++)

                {

                        System.out.println("     " + i + "         " + sales[i]);

                        sum += sales[i];

                }

 

            System.out.println("\nTotal-Dollar-Amount : " + sum);

    }

}
